\section{Key Contributions}
The main contributions of the thesis are expected to include:

\begin{itemize}

    \item a practical implementation of five comparable RAG pipelines;
    \item a local benchmark comparison of those pipelines;
    \item a reproducible workflow designed so the same files can be executed on HPC;
    \item a combined timing and output-quality evaluation framework; and
    \item a final local-versus-HPC comparison
\end{itemize}
